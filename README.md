# Read Me

    FarmHash, by Geoff Pike

    https://github.com/google/farmhash

    Provides a 128-bit hash 'Fingerprint128' equivalent to 'CityHash128 (v1.1.1)',
    which itself is equivalent to Google's 'FarmHash (v1.1)' as per GitHub repo.
